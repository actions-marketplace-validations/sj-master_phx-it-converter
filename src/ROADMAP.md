# Roadmap

The planned evolution of PHX-IT Converter. Open to community feedback and feature requests.

---

## Q3 2026 (July – September)

### v1.1 — Quality of Life
- [ ] SVG source file support — convert vector graphics to raster icons
- [ ] Custom size input — type any dimensions instead of picking from presets
- [ ] Keyboard shortcuts — quick convert, download all, clear
- [ ] Tooltip explanations for each format's use case

### v1.2 — Power User
- [ ] Multi-image batch upload — drop multiple images at once, convert all
- [ ] Preset save/load — save your favorite format combinations
- [ ] Conversion history — last 10 conversions saved in localStorage
- [ ] Export settings as shareable URL

---

## Q4 2026 (October – December)

### v1.3 — Developer Tools
- [ ] Favicon manifest generator — generate `site.webmanifest` alongside icons
- [ ] HTML snippet output — copy-paste ready `<link>` tags for your website
- [ ] Favicon checker — test existing favicons on any URL
- [ ] Image optimization pass — optional compression before conversion

### v1.4 — Polish
- [ ] PWA support — install as a desktop/mobile app, works offline
- [ ] Dark/light theme toggle with system preference detection
- [ ] Animation preferences — respect `prefers-reduced-motion`
- [ ] Accessibility audit and improvements
- [ ] Localization — Spanish and Japanese translations

---

## 2027

### v2.0 — Community & Growth
- [ ] User accounts (optional) — save presets and history across devices
- [ ] Public API — programmatic icon conversion for developers
- [ ] Icon template library — pre-designed icon shapes and styles
- [ ] Integration plugins — Figma, VS Code, WordPress
- [ ] Image upscaling — AI-powered icon enhancement for low-res sources
- [ ] Animated favicon support — GIF to animated ICO conversion

---

## Ideas & Backlog

The following ideas are under consideration but not yet scheduled:

- Watermark/overlay tool — add text or logo overlays to icons
- Background removal — auto-remove backgrounds for transparent icons
- Color palette extraction — generate icon sets with consistent color themes
- Icon comparison view — side-by-side before/after at multiple sizes
- CLI tool — `npx phx-it convert image.png --formats ico,png`
- Browser extension — right-click any image to convert
- Collaboration — share icon sets via temporary links
- AI icon suggestions — describe an icon and get AI-generated placeholder
